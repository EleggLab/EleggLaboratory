# App Two Research Plan (Milestone 3)

## Goal
자동매매 앱의 실거래 전 단계로, 전략 비교 + 백테스트 품질을 높인다.

## Scope
1. 전략 3종 운영 (SMA/RSI/Breakout)
2. 시나리오 3종 (상승/하락/횡보)
3. 리스크 파라미터 조절
4. 백테스트 결과 지표/주문 로그 가시화

## Out of Scope
- 실브로커 API 실제 주문
- 포트폴리오 멀티자산 최적화

## Exit Criteria
- 전략/시나리오 조합을 UI에서 바꾸고 백테스트 가능
- analyze/test/run 통과
