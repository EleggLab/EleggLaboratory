# Balance Reference Framework (2026-03-12)

목표: 현재 프로토타입(캐릭터 생성 + 생존 루프 + 오아시스 퀘스트)에 대해, 레퍼런스 게임의 밸런스 철학을 반영한 난이도 구조를 명확히 정의한다.

## 레퍼런스 축

- Fallout: New Vegas (Hardcore)
  - 자원 소비(물/식량) 압박이 전투 외 시간에도 누적되는 구조
- This War of Mine
  - 생존 이득과 도덕적 비용(인간성 손실)의 교환 관계
- Darkest Dungeon
  - 정신/심리 자원은 잃기 쉽고 회복은 제한적인 비대칭 구조
- S.T.A.L.K.E.R. 계열
  - 방사능 피폭이 장기 생존 리스크로 누적되는 구조
- Neo Scavenger
  - 하드 난이도에서 자원 부족과 회복 제한을 강하게 체감시키는 구조

## 현재 적용된 프로필

`src/crimson_wasteland/gameplay.py`의 `BalanceProfile`로 3단계 난이도를 운영한다.

| 프로필 | 의도 | 핵심 수치 |
| :--- | :--- | :--- |
| 스토리 | 진입 장벽 완화, 서사 진행 우선 | 방사능 75%, 적 피해 80%, 캡 보상 115%, 인간성 회복 120%/손실 80% |
| 스탠다드 | 기존 프로토타입 기준선 유지 | 모든 배율 100% 기준 |
| 하드코어 | 자원/피폭/전투 압박 강화 | 방사능 130%, 적 피해 125%, 캡 보상 90%, 인간성 회복 85%/손실 130% |

## 수식(코드 반영 기준)

- 탐험 방사능:
  - `적용 피폭 = round(기본 피폭 * 방사능배율 * (100 - 방사능저항)/10000)`
- 캡 보상:
  - `실보상 = round(기본 캡 * 캡배율/100)`
- 인간성 변화:
  - `회복 = round(기본 회복 * 회복배율/100)`
  - `손실 = round(기본 손실 * 손실배율/100)`
- 전투 적 피해:
  - `실피해 = round(기본 적피해 * 적피해배율/100)`

## 이번 사이클 반영 범위

- 밸런스 프로필 3종 구현
- 캠프 메뉴에서 프로필 즉시 전환
- 탐험/전투/정착지/퀘스트의 캡·인간성·방사능 변화에 난이도 배율 적용
- 평판 단계에 따른 정착지 상호작용 차등화(거래가/거래 차단/지원 효율)
- 성장 루프 1차(XP/레벨/퍼크포인트)와 밸런스 연계
- 테스트 3건 추가:
  - 하드코어 탐험 압박 증가 검증
  - 스토리 보상/패널티 완화 검증
  - 캠프 프로필 전환 검증
