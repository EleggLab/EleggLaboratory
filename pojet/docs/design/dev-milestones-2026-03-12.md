# Pojet 개발 마일스톤 (실행본)

작성일: 2026-03-12
기준: 재미 + 강한 성인 톤 + 1로그 1선택 체감 강화

## M1. 성인 톤 강도 계층 (S1~S5)
목표: 로그/이벤트가 상황별로 성인 톤 강도를 일관되게 유지하도록 엔진화.
완료 조건:
- 상태 기반 톤 강도 계산기(S1~S5) 구현
- 로그 메타(`maturity:*`)에 강도 태그 반영
- 이벤트 선택 가중치에 톤 강도 반영

## M2. 선택 즉시 보상 + 지연 보상 루프
목표: 선택 직후 체감 + 몇 틱 뒤 후속 결과를 모두 제공.
완료 조건:
- 선택 직후 기존 즉시 patch 유지
- 지연 결과 큐(`deferred outcomes`) 추가
- due tick 도달 시 자동 적용 + 로그 출력

## M3. 분기/반복 안정화 강화
목표: 비슷한 이벤트 연속 노출을 더 줄이고, 분기 피로를 완화.
완료 조건:
- 톤 강도/카테고리/최근 선택 히스토리 기반 가중치 보정
- QA에서 인접 동일 이벤트 비율 목표치 유지

## M4. QA 게이트 (100회+)
목표: 자동 시뮬레이션 기반으로 기능 안정성 검증.
완료 조건:
- 120회 이상 QA 스트레스 테스트 PASS
- 실패 샘플 0 또는 임계치 이하

## 이번 턴 적용 범위
- M1 구현
- M2 구현
- M3 일부 보강
- M4 120회 QA 실행

## 진행 현황 (이번 턴 결과)
- M1: 완료
  - 상태 기반 S1~S5 계산기 추가
  - 로그 태그(`maturity:Sx`, `tone-style:*`) 반영
  - 이벤트 선택 가중치에 강도 반영
- M2: 완료
  - 선택 후 지연 결과 큐(`deferredOutcomes`) 추가
  - due tick 도달 시 자동 적용 + `[후속]` 로그 생성
- M3: 완료(1차)
  - 이벤트 ID 외에 카테고리/티어 반복 패널티 추가
  - 고강도 상태에서 성인 카테고리 선호 가중치 강화
- M4: 완료
  - QA Stress Test `140 runs x 50 steps` 통과
