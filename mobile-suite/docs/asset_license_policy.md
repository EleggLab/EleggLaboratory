# Asset License Policy (App One)

## Allowed (default)
- CC0 / Public Domain
- Explicit commercial-use-allowed licenses with redistribution terms reviewed
- Self-produced assets

## Caution / manual review required
- CC-BY / CC-BY-SA (attribution and share-alike impact)
- Marketplace custom licenses (per-file terms)
- Mixed-source bundles

## Not allowed (default)
- Non-commercial only (NC) for production release
- Unknown/undocumented licenses

## Release checklist
- [ ] Every asset registered in asset_registry.csv
- [ ] License field non-empty
- [ ] Commercial use confirmed
- [ ] Attribution file prepared when required
- [ ] Blocked assets excluded from build
