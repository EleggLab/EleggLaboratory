# Pojet 초심자 시점 점검 (야겜 인지성 검토)

작성일: 2026-03-12
가정: 유저는 사전 지식 없이 "야겜"이라는 정보만 알고 처음 실행.

## 핵심 문제 (점검 결과)

1. 초반 1~3분에 야겜 정체성 신호가 약함
- 튜토리얼 문구가 시스템 중심이라 장르 톤이 늦게 드러남.

2. 이벤트 카드에서 성인 분기인지 즉시 식별이 어려움
- 기존 메타는 `수동/자동 처리` 정보만 보여줌.

3. 초반 이벤트가 일반 탐험/전투로만 이어질 가능성 존재
- 유저가 "야겜 맞나?"라고 느낄 수 있는 구간 발생.

## 이번 턴 적용 개선

1. 튜토리얼 텍스트를 장르 중심으로 교체
- 18+ 다크판타지, 욕망/권력/대가, 관계/오염 시스템을 첫 화면에서 명시.

2. 런 시작 직후 세계관/규칙 로그 추가
- 성인 다크판타지 진입 안내 + 관계/오염 수치 반영 구조를 즉시 고지.

3. 초반 아이덴티티 강제 분기 추가
- 오픈 이벤트 3개 이내에 성인 카테고리 이벤트가 없으면 T2에서 성인 카테고리 분기를 강제 선택.

4. 이벤트 카드 메타 강화
- `성인 톤: ...` 또는 `톤: ...` 표시로 이벤트 성격을 즉시 인지 가능.

## 잔여 과제 (다음 개발)

1. 온보딩에 `성인 콘텐츠 경고/선호 강도(S1~S5)` 설정 UI 추가
2. 초반 고정 시나리오(프롤로그 3로그)로 장르 훅 강화
3. 성인 루트/전술 루트 선택 시 기대 결과를 선택지에 더 명확히 노출
4. 유저 리서치: 첫 10분 "야겜 인지 여부" 설문 지표 수집
