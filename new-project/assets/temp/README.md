## Included
- `bg_wasteland_dawn.svg`: 오프닝/탐험용 배경
- `bg_ruins_night.svg`: 위기/야간 이벤트용 배경
- `portrait_scavenger.svg`: 생존자 계열 인물 포트레이트
- `portrait_medic.svg`: 의무관/지원형 인물 포트레이트
- `icon_radiation.svg`: 방사능 스탯 아이콘
- `icon_hunger.svg`: 허기 스탯 아이콘

- `ui/index.html`에서 현재 파일들을 직접 참조합니다.
- 최종 아트 에셋 확정 시 동일 파일명으로 교체하면 참조 유지가 쉽습니다.
