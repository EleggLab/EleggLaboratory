## 성인 다크판타지 톤 제작 팁
1. 배경: 채도 낮은 중간톤 + 포인트 색 1개(적/보라)
2. UI: 텍스처 노이즈 + 얇은 장식선(과장 금지)
3. 초상: 표정 레이어(중립/경계/유혹/분노/피로) 분리
4. 조명: day/night 두 장만 먼저 만들고 이벤트별 오버레이로 확장
